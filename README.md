# Калькулятор Бассейна

Веб-приложение для расчета материалов и стоимости строительства бассейна.

## Функциональность

- Расчет необходимых материалов на основе размеров бассейна
- Поддержка двух типов бассейнов: с пленкой ПВХ и с керамической плиткой
- Расчет площадей (дно, стены, ступени)
- Редактор норм расхода материалов
- Экспорт результатов в Excel и PDF

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements_web.txt
```

2. Запустите приложение:
```bash
python -m flask --app src/web/app.py run
```

## Использование

1. Введите размеры бассейна (длина, ширина)
2. Выберите тип бассейна (ПВХ или керамика)
3. Укажите количество ступеней
4. Нажмите "Рассчитать"

## Редактор норм расхода

Для изменения норм расхода и цен на материалы:
1. Нажмите "Редактор норм расхода"
2. Измените нужные значения
3. Нажмите "Сохранить"

## Экспорт

- Excel: нажмите "Экспорт в Excel"
- PDF: нажмите "Экспорт в PDF"
